# 항해99 주특기 심화과정

리액트 심화과정 내용

## 기술스택
- React & Firebase
- 사용패키지:
redux & redux thunk & redux-logger & redux-actions
immer, history, connected-react-router
moment, lodash, react-router-dom
css : styled-components & MUI

## 구현

회원
- 로그인기능 (세션 & 쿠키)
- 회원가입기능 (이메일, 비밀번호, 패턴구현)
- 로그아웃기능 (세션 & 쿠키삭제)
- 
SNS/블로그 (CRUD)
- 게시글 읽기
- 게시글 추가
- 게시글 수정
- 게시글 삭제
- 디테일 화면 구현
- 좋아요 기능(실시간 변화 o)
- 댓글기능
- 알람기능 (realtime database)
- 무한스크롤
